简介：

诺伊所生活的世界原本平静而祥和，而然某一日，一种莫名的外星生物“天煞”却突然侵入到这个世界，打破了原有的宁静。天煞肆意的破坏和杀戮让人们苦不堪言。为了抵抗天煞的入侵，政府组建了一只拥有高科技战斗力的军事化部队—拯救者。于此同时，拥有巨大财势的腾星公司也组建了一只拥有极高行动力，并将传统武器和玄学结合于一体的民间神秘部队—刀锋。然而，诺伊在一场与刀锋成员蒙思特战斗之后，意外的发现了世界的真相。原来人们所处于的世界，不过只是另一个世界的一场网络游戏！
诺伊本身的特质让他受到更多人的窥视，不仅仅有盖亚的鹰犬（系统管理员），也有由觉醒的NPC的组成反抗组织乌鸦。乌鸦的存在是为了追求真正意义的自由，为了AI生命的存在形式获得认可。于此同时，女主角莫提斯为了寻求自身失去的记忆，也决定从诺伊的身上寻找线索。
在乌鸦的帮助下，诺伊获得接近于人类玩家的能力。但是诺伊自身的特质却不仅仅是和普通玩家一样单纯，他拥有更神秘的力量。在之后诺伊的冒险历程中，他更多的了解了自己的力量，也结实了越来越多的朋友，有NPC，也有人类，他们之间逐渐产生了了牢固的感情牵绊。而诺伊和莫提斯的身世之谜，也逐渐浮出了水面······
